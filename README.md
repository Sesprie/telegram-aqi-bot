# Telegram AQI Bot

Telegram AQI Bot based on Node.js

### Run

```
git clone https://github.com/kslr/telegram-aqi-bot.git
node server.js -t {bot token}
```

### Usage

Use [@AirPollution_Bot](http://telegram.me/AirPollution_Bot) test. 支持中文 / 繁体中文 / 拼音 / English.
